深大集市 — 校园特色时令农产品交易平台

为深圳大学师生提供便捷、安全的农特产交易服务

2024年12月19日 - 2025年1月2日
个人全栈项目
Go + HTML/CSS/JS + MySQL

项目简介

深大集市是一个面向深圳大学校园的B2C时令农产品在线交易平台，采用前后端分离架构，旨在为校内师生提供便捷、安全的农特产交易服务。

该项目由我独立设计并开发，完整实现了商品展示、在线交易、用户管理、后台审核等核心电商功能。平台上线两周内吸引了500+用户注册，完成200+笔交易。

项目目标：解决校园内季节性农产品交易不便的问题，连接本地农户与学生群体，打造校园特色电商生态。

技术栈与实现

前端技术

  • 原生 HTML5 + CSS3 + JavaScript (ES6+)
  • 响应式设计，适配多终端设备
  • 性能优化：图片懒加载、模块化组件
  • 用户体验：动态粒子效果、平滑过渡

后端技术

  • Go 语言 (net/http 原生实现)
  • RESTful API 设计规范
  • MySQL 数据库 + go-sql-driver
  • 模块化分层架构 (MVC 模式)

关键模块

  • 用户系统：注册/登录/会话管理
  • 权限控制：RBAC 两级权限系统
  • 商品管理：审核/上下架/违规处理
  • 交易流程：购物车/订单/支付
  • API 安全：参数验证/SQL 注入防护
